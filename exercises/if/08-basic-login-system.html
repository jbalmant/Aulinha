8. Sistema de login simples:

Você está desenvolvendo um sistema de login.

Escreva um programa que solicite ao usuário seu nome de usuário e senha. O acesso será concedido se o nome de usuário for "admin" e a senha for "senha123", ou se o nome de usuário for "user" e a senha for "userpass". Em todos os outros casos, o acesso será negado.

O programa deve imprimir mensagens diferentes para cada caso: "Acesso concedido" se as credenciais estiverem corretas e "Acesso negado" caso contrário..
